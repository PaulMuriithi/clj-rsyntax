API for clj-rsyntax.rtextarea - clj-rsyntax

by Jon Rose
Full namespace name: clj-rsyntax.rtextarea

Overview

Wrapper API for org.fife.ui.org.fife.ui.rtextarea.RTextArea.org.fife.ui.rtextarea.RTextArea (http://fifesoft.com/) the
	opensource text editor  component from RText.
